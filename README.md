# Forensics-Calendar
电子数据取证赛事日历

## json生成

- 赛事日程：https://forensics-wiki.github.io/Forensics-Calendar/calendar.html
- Writeup: https://forensics-wiki.github.io/Forensics-Calendar/writeup.html

## 添加赛事
1. 如需添加赛事，请进入`index.html`页面
2. 按照内容填写，并生成json数据
3. fork本项目，修改`Calendar.json`后pr即可
